# BaDaaS Typography

## Primary Font

**Source Code Pro**

- Type: Monospace
- Weight: Variable (100-900)
- License: SIL Open Font License
- Source: [Google Fonts](https://fonts.google.com/specimen/Source+Code+Pro)

### Usage

Used for all text on the website including headings and body text. The monospace
style reinforces the technical/cryptography theme.

### CSS

```css
font-family: "SourceCodePro", "SF Mono", Monaco, "Andale Mono", monospace;
```

## Display Font (Optional)

**VT323**

- Type: Monospace, retro terminal style
- Weight: 400
- License: SIL Open Font License
- Source: [Google Fonts](https://fonts.google.com/specimen/VT323)

### Usage

Available for special display purposes where a retro terminal aesthetic is
desired. Not currently used in the main design.

## Font Files

Font files are located in `/public/fonts/`:

- `SourceCodePro-VariableFont_wght.ttf`
- `VT323-Regular.ttf`
