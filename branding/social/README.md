# Social Media Assets

Pre-sized assets for social media platforms.

## Required Sizes

| Platform    | Asset   | Size (px) | Status |
| ----------- | ------- | --------- | ------ |
| Twitter/X   | Profile | 400x400   | [x]    |
| Twitter/X   | Header  | 1500x500  | [x]    |
| LinkedIn    | Profile | 400x400   | [x]    |
| LinkedIn    | Banner  | 1584x396  | [x]    |
| GitHub      | Profile | 500x500   | [x]    |
| Open Graph  | Default | 1200x630  | [x]    |
| Favicon     | Icon    | 32x32     | [x]    |
| Apple Touch | Icon    | 180x180   | [x]    |

## File Naming

```
{platform}-{type}.png

Examples:
- twitter-profile.png
- linkedin-banner.png
- og-image.png
```

## Notes

- All images should be PNG with transparency where appropriate
- Use the accent color (#c1bb3a) on dark background (#191919)
- Ensure logo is centered with adequate padding
